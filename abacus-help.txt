Usage: abacus [OPTIONS] COMMAND [ARGS]...

Options:
  --help  Show this message and exit.

Commands:
  chart   Define chart of accounts.
  ledger  Post entries to ledger.
  report  Show reports.
Usage: abacus chart [OPTIONS] COMMAND [ARGS]...

  Define chart of accounts.

Options:
  --help  Show this message and exit.

Commands:
  add        Add account to chart.
  add-many   Add several accounts to chart.
  init       Create chart of accounts file (chart.json) in current folder.
  name       Change account title.
  operation  Define a named pair of debit and credit accounts.
  set        Change default names of retained earnings, income summary or...
  show       Print chart of accounts.
  unlink     Permanently delete chart of accounts file in current folder.
Usage: abacus ledger [OPTIONS] COMMAND [ARGS]...

  Post entries to ledger.

Options:
  --help  Show this message and exit.

Commands:
  close           Close ledger at accounting period end.
  init            Create ledger file (entries.linejson) in current folder.
  post            Post double entry to ledger.
  post-compound   Post compound entry to ledger.
  post-operation  Post operation to ledger.
  show            Print ledger.
  unlink          Permanently delete ledger file in current folder.
Usage: abacus report [OPTIONS] COMMAND [ARGS]...

  Show reports.

Options:
  --help  Show this message and exit.

Commands:
  account           Show or check account information.
  balance-sheet     Show balance sheet.
  balances          Show account balances.
  income-statement  Show income statement.
  trial-balance     Show trial balance.
